# spgp_2024

# 게임 컨셉<br/>
High Concept -> 파이프를 회전시켜 하나의 연결된 파이프를 완성하여 스테이지를 클리어 합니다.<br/>
핵심 메카닉 -> 플레이어는 클릭과 회전을 통해 파이프를 연결합니다<br/>

# 개발 범위
<br/>8개 이상의 스테이지<br/>
