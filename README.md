GridLayout, Toast messages, Handler 등 <br/>

- 리소스 수정
- 스테이지 갯수 늘리기
- 추가적인 UI
- 테마 등의 인앱결제 항목
- 실행을 시킬때 컴퓨터가 아에 멈추어서 강제종료 하는 일이 종종 생기는데 아직도 해결하지 못했습니다,,
